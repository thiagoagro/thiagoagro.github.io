# Thiago Orlando Costa Barboza — Digital Agriculture Portfolio

Static personal portfolio for GitHub Pages, built with plain HTML, CSS and JavaScript.

## Sections

- **About** — hero section, research profile, metrics and carousel timeline.
- **Experience** — education, research experience, skills, lectures and memberships.
- **Papers** — filterable list of journal papers, conference publications and book chapters.
- **Projects** — repository cards connected to digital agriculture applications.

## How to edit

Most content is centralized in:

```txt
assets/js/data.js
```

Edit the arrays named `timeline`, `experience`, `skills`, `activities`, `publications` and `projects`.

To replace the profile photo, overwrite:

```txt
assets/images/profile.jpg
assets/images/avatar.jpg
```

## How to publish on GitHub Pages

1. Upload these files to the root of your repository.
2. Keep the `.nojekyll` file in the root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose branch `main` and folder `/ (root)`.
6. Save and wait for GitHub Pages to deploy.
